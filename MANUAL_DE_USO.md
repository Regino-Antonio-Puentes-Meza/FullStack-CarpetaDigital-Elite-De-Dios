## 📋 INFORMACIÓN GENERAL

Este proyecto es una aplicación web portable del **Archivo Digital del Santuario Elíte de Dios** que puede ejecutarse como un archivo `.exe` sin necesidad de instalación adicional.

### 🚀 CARACTERÍSTICAS PRINCIPALES
- ✅ **Ejecutable portable**: No requiere instalación
- ✅ **Auto-arranque**: Abre automáticamente el navegador
- ✅ **Galería de fotos**: 112+ testimonios sagrados organizados
- ✅ **Documentos PDF**: Acceso directo a información espiritual y documentación
- ✅ **Seguridad**: Versión encriptada con AES-256-CBC
- ✅ **Compatibilidad**: Funciona en Windows 10/11

## 🔧 INSTRUCCIONES DE USO

### Instalación
1. **No se requiere instalación**
2. Simplemente descarga el archivo `.exe` deseado
3. Colócalo en cualquier carpeta de tu computadora

### Ejecución
1. **Haz doble clic** en el archivo `.exe`
2. **Espera** a que aparezca la consola (2-3 segundos)
3. **El navegador se abrirá automáticamente** con la aplicación
4. **¡Listo!** Ya puedes navegar por la carpeta digital

### Cerrar la aplicación
- **Método 1**: Presiona `Ctrl+C` en la consola
- **Método 2**: Cierra la ventana de la consola
- **Método 3**: Cierra directamente el navegador

## 🛠️ SOLUCIÓN DE PROBLEMAS

### El ejecutable no inicia
1. **Verifica antivirus**: Agrega excepción si es necesario
2. **Ejecuta como administrador**: Click derecho → "Ejecutar como administrador"
3. **Windows Defender**: Permitir aplicación si aparece advertencia

### El navegador no se abre automáticamente
1. **Revisa la consola**: Busca la URL (generalmente `http://localhost:3000`)
2. **Abre manualmente**: Copia la URL y pégala en tu navegador
3. **Puerto ocupado**: La aplicación buscará otro puerto automáticamente

### Las fotos no cargan
1. **Verifica conexión**: Asegúrate de que el servidor esté funcionando
2. **Revisa la consola**: Busca mensajes de error
3. **Refresca la página**: Presiona F5 en el navegador

### Los PDFs no abren
1. **Lector PDF**: Asegúrate de tener un lector PDF instalado
2. **Navegador actualizado**: Usa una versión reciente del navegador
3. **Bloqueo de pop-ups**: Permite ventanas emergentes para el sitio
